# Caffelo - Responsive Coffee Shop Website
A modern, high-end, and fully responsive coffee shop website built using Bootstrap 5. This project, developed in 2023 for practice, demonstrates the integration of a popular frontend framework with custom CSS to create a professional business landing page.

# Key Features
Bootstrap 5 Framework: Utilizes Bootstrap’s powerful grid system and components for a mobile-first design.

Fully Responsive: Seamlessly adapts to all screen sizes, including desktops, tablets.

Sticky Navigation Bar: A functional, branded navbar that allows easy access to Home, About, Menu, Products, Gallery, and Contact sections.

Interactive Menu & Gallery: Organized sections showcasing products and shop highlights with hover effects.

Custom Aesthetics: Enhanced with specialized CSS for scrollbar styling, button transitions, and a "Back to Top" navigation feature.

Professional Typography: Features a clean look using Merriweather and Roboto fonts from Google.

# Tech Stack
HTML5 & CSS3: For semantic structure and custom theme styling.

Bootstrap 5: For the layout framework and responsive utilities.

Font Awesome 6: For high-quality vector icons (Social media, contact info).

Google Fonts: Integrated typography for a premium feel.

# File Structure
coffee shop.html: The core structure containing the Bootstrap-based sections.

style.css: Custom styling for brand identity, hover animations, and scroll behaviors.

# Background
This project was created in 2023 as a technical practice to master Bootstrap 5. It focuses on building a cohesive brand identity (Caffelo) while ensuring the website remains functional and fast across various devices.
